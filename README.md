# CodeAlpha_CarPricePrediction
Car Price Prediction using Machine Learning

## Objective
Predict the selling price of used cars using machine learning.

## Algorithm
Random Forest Regressor

## Results
- MAE: 0.602
- MSE: 0.797
- R² Score: 0.965

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
